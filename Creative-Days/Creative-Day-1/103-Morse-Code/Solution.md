## Code
[103-Morse-Code.ino](103-Morse-Code.ino)

In our code:
- Switch 1 spells out SOS
- Switch 2 spells out TTT (another distress signal)
- Switch 3 spells out HELP
- When all switches are on, each message plays in sequence.
